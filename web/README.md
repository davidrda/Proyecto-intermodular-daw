# Ragnarok Gym — Portal Web

Portal web corporativo del gimnasio ficticio Ragnarok Gym Center. Forma parte del Proyecto Intermodular de 1º DAW y se corresponde con los módulos de **Lenguajes de Marcas** y el desarrollo frontend del proyecto.

El sitio está hecho únicamente con **HTML y CSS**, sin frameworks ni JavaScript. El objetivo es practicar maquetación semántica, Flexbox, Grid y variables CSS, además de definir la identidad visual del proyecto que luego se mantiene en la aplicación Java y el resto de entregables.

## Páginas

El portal tiene tres páginas:

- **`index.html`** — Página principal con presentación del gimnasio, servicios, tarifas y testimonios.
- **`nosotros.html`** — Información sobre el gimnasio, su filosofía y sus instalaciones.
- **`contacto.html`** — Formulario de contacto y datos para localizar el gimnasio.

## Estilo visual

La estética es **oscura e industrial**, con negro y grises metálicos como base y un amarillo-verde neón (`#f0f000`) como color de acento. La tipografía combina **Bebas Neue** para titulares impactantes con **Barlow** para el cuerpo de texto.

Los colores y las fuentes están definidos como **variables CSS** en `:root`, lo que permite cambiarlos fácilmente desde un único sitio.

## Estructura

```
/web
├── index.html          → página principal
├── nosotros.html       → sobre el gimnasio
├── contacto.html       → formulario de contacto
├── styles.css          → hoja de estilos
└── img/                → imágenes del portal
```

## Tecnologías

- HTML5 (maquetación semántica)
- CSS3 (Flexbox, Grid, variables CSS)
- Google Fonts (Bebas Neue, Barlow)

## Cómo visualizarlo

Basta con abrir `index.html` en cualquier navegador. No hace falta servidor ni instalación.
